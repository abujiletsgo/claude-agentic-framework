How It Works

Every project gets its own .mempalace/ directory with a ChromaDB vector store and SQLite knowledge graph. Agent results are automatically stored and injected across sessions.

Store (SubagentStop)
subagent_palace_store.py
Captures agent output in ChromaDB drawer
Extracts decisions to KG triples
ASYNC ~994ms (non-blocking)
Inject (SubagentStart)
subagent_kg_inject.py
Queries project-local KG for decisions
Injects as additionalContext
~89ms, ~222 tokens per agent
MCP Access (On-Demand)
Researcher agent
mempalace_search + mempalace_kg_query
Deep search when injection isn't enough
Zero API token cost

Key Design Decisions

DecisionRationale
Project-local .mempalace/Each project gets its own ChromaDB + KG. Prevents cross-project contamination. Global memory stays short.
KG-only inject (not ChromaDB search)SQLite KG queries are sub-ms. ChromaDB search adds ~200ms. Fast path keeps SubagentStart under 100ms.
Async store on SubagentStopChromaDB storage takes ~1s (subprocess + embedding). Async means zero blocking on agent completion.
Fail-open everywhereIf mempalace is unavailable, all hooks emit {} and exit 0. Framework works without mempalace installed.
Subprocess for ChromaDBChromaDB needs numpy compiled for Python 3.12 (mempalace venv), but hooks run on system Python 3.14. Subprocess bridges the version gap.

Subsystems

Damage Control
100+ blocked patterns. Path protection (zero-access / read-only / no-delete). SHA-256 skill integrity verification. Prevents destructive commands, credential leaks, force-pushes to main.
Caddy Classifier
Classifies every prompt on 4 dimensions: complexity, type, quality, scope. Routes to: direct / orchestrate / rlm / fusion. Confidence scoring. Auto-delegation hints.
Knowledge Pipeline
SQLite FTS5 for cross-session learning. auto_fact_extractor writes verified facts. validate_facts prunes >90 day entries. CONFIRMED > GOTCHAS > PATHS > PATTERNS.
Mempalace NEW
Project-local ChromaDB + SQLite KG. Auto-stores agent results on SubagentStop. Injects KG decisions on SubagentStart (~222 tokens). Researchers get MCP search tools. Zero API token cost for all operations.
RepoMap
TreeSitter symbol index auto-generated for repos with 200+ source files. Ranked by PageRank-style algorithm. Injected at session start for large repos.
Circuit Breakers
If a hook fails 3 times, auto-disables for 60s. State in ~/.claude/circuit_breakers/. Prevents runaway hook execution. Delete state file to reset.
